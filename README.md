#Objetivo do Programa

O presente trabalho tem como objetivo consolidar os conceitos fundamentais de **lógica de programação estruturada em C++**, através da resolução de seis problemas práticos que envolvem:

- Manipulação de vetores (arrays) unidimensionais;
- Uso de vetores paralelos para armazenamento de dados relacionados;
- Cálculo de médias, percentagens, totais e descontos;
- Busca sequencial em vetores;
- Ordenação mantendo a integridade dos dados paralelos;
- Aplicação de condições e tomada de decisões.

O programa final é composto por seis módulos independentes, cada um resolvendo um problema específico conforme enunciado.
